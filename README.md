# HabitHive

HabitHive is a productivity web application that helps users manage tasks, track assignments, and maintain daily streaks.

## Features
- User Registration & Login
- Task Management
- Assignment Tracking
- Daily Goal Tracking
- Streak System

## Tech Stack
- Flask (Python)
- SQLite
- HTML, CSS
